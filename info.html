このサイトについて

  1. このサイトはメンバー用のものです。公開用のサイトはここ
  2. このサイトの運営はsonnekoが行なっています。何かあれば、sonnekoまでお願いしますす。
  3. このサイトはここから編集することができます。
  4. 編集はsg-amr-memberではなく、個人のアカウントで行なってください。
  5. index.html・news.html・info.htmlは編集しないでください。
  6. このサイトはメンバー以外の人が見ることができますので、パスワード・本名など秘匿すべきものは入力しないでください。
  7. book.html・to_do.html・schedule.htmlを編集してくれるとありがたいです。
    編集はここから
  8. このサイトのURLは公開サイトに漏らさないようにしてください。

公開用のサイトについて

  1. 公開用サイトは公式Xにリンクを貼っているので、公式Xの投稿内容と矛盾するような内容・個人情報などを書き込まないでください。
    もし、挿入してから削除したとしてもバージョン履歴として残ってしまうので、注意してください。
  2. index.html・info.html・comment.html・news.htmlは編集しないでください。
  3. 是非、member.htmlに自分の欄を追加して、情報を書き込んでください。ここから
  4. フォルダblogの中に、自分の名前(偽名)のフォルダを作成して、その中にそれぞれのブログを作ってくれても構いません。ここから
    ただし、内容はチームamorphousに準拠するものとしてください。

<!DOCTYPE>
<html>

<!--なお、利用するときはこのコメントを削除してください。-->

	<title>amorphous公式サイト</title>

	<link rel="stylesheet" href="css/base.css">
</head>
<body>
	<header>
		<h3>Amorufasu公式サイト</h3>
		<h1><!--ここに題名を書く--></h1>
	</header>
	<main>
		<!--ここから、htmlを書く-->
		
		<!--ここまで-->
	</main>
	<footer>
		<h4>サイトマップ</h4>
		<p><a href="index.html">ホーム</a></p>
		<h5><a href="blog.html">ブログ</a></h5>
		<!--フッターのサイトマップを全ページで共通にしたいので、外部ファイルから読み込む形をとりました。-->
		<!--ブログへのリンクを自動追加 #blog_linkにhtmlが追加されます。-->
		<script src="https://d3js.org/d3.v3.min.js" charset="utf-8"></script>
		<div id="blog_link"></div>
		<script>
		d3.csv("https://docs.google.com/spreadsheets/d/[スプレッドシートのid]/export?format=csv&range=A3:D", function(error, data){
		var text = "";
		for(var i=0; i<data.length; i++){
			text +=  data[i].Comments;
		}
		d3.select("#blog_link").html(text);
		});
		</script>
		<!--ブログへのリンクを自動追加終わり-->
		<br><p>
			<a href="info.html">Amorphousについて</a><br>
			<a href="member.html">メンバー紹介</a><br>
			<a href="news.html">お知らせ</a><br>
			<a href="comment.html">コメント機能</a>
		</p>
	</footer>
</body>
